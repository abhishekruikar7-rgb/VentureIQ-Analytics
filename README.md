# VentureIQ - Startup Investment Analytics Platform

## Overview

VentureIQ is a data analytics platform that analyzes startup performance, funding rounds, investor behavior, and risk factors using MySQL and Power BI.

## Problem Statement

Investors need a centralized system to evaluate startups based on:
- Funding received
- Growth rate
- Revenue
- Burn rate
- Investor preferences

## Tech Stack

- MySQL
- SQL
- Power BI
- DAX
- Data Modeling

## Database Design

Tables:

1. startups
2. founders
3. funding_rounds
4. startup_metrics
5. investors
6. investments


## Features

### Startup Overview Dashboard
- Total startups
- Total funding
- Industry distribution
- Startup growth analysis


### Risk Analysis Dashboard
- High-risk startup identification
- Revenue vs burn rate comparison
- Growth analysis


### Investor Portfolio Dashboard
- Investor contribution
- Funding distribution
- Investment trends


## Key SQL Concepts Used

- Joins
- Aggregations
- Group By
- Foreign Keys
- Relationships
- Filtering


## Business Impact

Helps investors identify:
- High-growth startups
- Risky investments
- Best funding opportunities
